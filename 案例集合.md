# 类数组
下面打印obj的结果是什么？
```javascript
var obj = {
    '2': 3,
    '3': 4,
    'length': 2,
    'splice': Array.prototype.slice,
    'push': Array.prototype.push
}


obj.push(1);
obj.push(2);

console.log(obj);
```
代码剖析
```javascript
//由上面可知obj是一个类数组，因为obj具有从零连续的属性集合(此时0，1为empty值)，第二个是有length属性
//当obj.push的时候发生了什么呢，下面是push原理代码
Array.prototype.push = function(elem) {
    this[this.length] = elem;
    this.length++
}


```
